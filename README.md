# Before we go
This is my first time I've used:

 - Laravel
 - Git
 - OAuth based API


# Requirements

PHP 7.1.3+


# Installation

 - Clone 
 - composer install
 - Setup env. Assign these values:

>  GITHUB_API_URL  
>  GITHUB_DEFAULT_REPO  
>  GITHUB_CLIENT_ID 
> GITHUB_CLIENT_SECRET

 - php artisan config:cache
